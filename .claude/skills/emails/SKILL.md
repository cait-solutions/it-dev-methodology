## Основные принципы

### 1. Одно письмо — одна задача
- Каждое письмо имеет одну основную цель
- Один основной CTA на письмо
- Не пытаться сделать всё

### 2. Ценность до просьбы
- Вести с полезностью
- Строить доверие через контент
- Заработать право продавать

### 3. Релевантность важнее объёма
- Меньше лучших писем выигрывает
- Сегментировать для релевантности
- Качество > частота

### 4. Чёткий путь вперёд
- Каждое письмо куда-то ведёт
- Ссылки должны что-то делать полезное
- Следующие шаги должны быть очевидны

## Стратегия email-последовательности

### Длина последовательности
- Welcome: 3-7 писем
- Lead nurture: 5-10 писем
- Onboarding: 5-10 писем
- Re-engagement: 3-5 писем

Зависит от:
- Длины sales цикла
- Сложности продукта
- Стадии отношений

### Timing/задержки
- Welcome письмо: Немедленно
- Ранняя последовательность: 1-2 дня между
- Nurture: 2-4 дня между
- Долгосрочная: Еженедельно или раз в 2 недели

### Стратегия тем писем
- Чётко > Остроумно
- Конкретно > Расплывчато
- Ориентировано на выгоду или любопытство
- 40-60 символов идеально

**Паттерны которые работают:**
- Вопрос: "Всё ещё борешься с X?"
- Как сделать: "Как [достичь результата] за [время]"
- Число: "3 способа [выгоды]"
- Прямо: "[Имя], твой [объект] готов"
- Тизер истории: "Ошибка которую я сделал с [темой]"

## Типы последовательностей

### Welcome-последовательность (после регистрации)
**Длина**: 5-7 писем за 12-14 дней
**Цель**: Активация, построение доверия, конверсия

Ключевые письма:
1. Welcome + доставить обещанную ценность (немедленно)
2. Быстрая победа (день 1-2)
3. История/Почему (день 3-4)
4. Социальное доказательство (день 5-6)
5. Преодоление возражения (день 7-8)
6. Highlight ключевой функции (день 9-11)
7. Конверсия (день 12-14)

### Lead Nurture-последовательность (до продажи)
**Длина**: 6-8 писем за 2-3 недели
**Цель**: Построить доверие, демонстрировать экспертизу, конвертировать

Ключевые письма:
1. Доставить lead magnet + intro (немедленно)
2. Расширить тему (день 2-3)
3. Глубокое погружение в проблему (день 4-5)
4. Фреймворк решения (день 6-8)
5. Кейс (день 9-11)
6. Дифференциация (день 12-14)
7. Обработчик возражений (день 15-18)
8. Прямое предложение (день 19-21)

### Re-engagement последовательность
**Длина**: 3-4 письма за 2 недели
**Триггер**: 30-60 дней неактивности
**Цель**: Вернуть или очистить список

Ключевые письма:
1. Check-in (искренняя забота)
2. Напоминание о ценности (что нового)
3. Стимул (специальное предложение)
4. Последний шанс (остаться или отписаться)

### Onboarding-последовательность (пользователи продукта)
**Длина**: 5-7 писем за 14 дней
**Цель**: Активация, ведение к aha moment, upgrade
**Примечание**: Координировать с in-app онбордингом — email поддерживает, не дублирует

Ключевые письма:
1. Welcome + первый шаг (немедленно)
2. Помощь с началом (день 1)
3. Highlight функции (день 2-3)
4. История успеха (день 4-5)
5. Check-in (день 7)
6. Продвинутый совет (день 10-12)
7. Upgrade/расширение (день 14+)

## Типы писем по категории

### Onboarding письма

- Переход на годовую подписку
- Восстановление при ошибке платежа
- Опрос при отмене
- Напоминания о предстоящем продлении

- Ежедневные/еженедельные/ежемесячные сводки
- Уведомления о ключевых событиях

## Рекомендации по email-копи

### Структура
1. **Hook**: Первая строка захватывает внимание
2. **Контекст**: Почему это важно для них
3. **Ценность**: Полезный контент
4. **CTA**: Что делать дальше
5. **Sign-off**: Человеческое, тёплое закрытие

### Форматирование
- Короткие абзацы (1-3 предложения)
- Пустое место между секциями
- Bullet points для scannability
- **Жирный** для акцента (редко)
- Mobile-first (большинство читают на телефоне)

### Тон
- Разговорный, не формальный
- От первого лица (я/мы) и второго (ты)
- Активный залог
- Прочитай вслух — звучит ли по-человечески?

### Длина
- 50-125 слов для транзакционных
- 150-300 слов для образовательных
- 300-500 слов для narrative

### Рекомендации по CTA
- Кнопки для основных действий
- Ссылки для вторичных действий
- Один чёткий основной CTA на письмо
- Текст кнопки: Действие + результат
